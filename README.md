# Getting Started

## API

`$ cd api`

`$ python3 -m venv env`

`$ source env/bin/activate`

`$ pip3 install -r requirements.txt`

`$ ./manage.py runserver 0.0.0.0:8000`

## Mobile

`$ make install`

`$ make run`
